# 🌍 Cross-Border Recruiter Agent

An AI-powered recruitment system that automates candidate sourcing, screening, and ranking across different regions with cultural intelligence.

---

## 🚀 Overview

This project simulates a real-world recruiter workflow by:
- Parsing job descriptions
- Sourcing candidates from GitHub & LinkedIn
- Evaluating profiles using a multi-factor scoring model
- Ranking candidates automatically
- Generating detailed PDF reports

---

## 🧠 Key Feature: Cultural Intelligence

Unlike traditional systems, this agent evaluates **regional hiring preferences**:

- 🇸🇦 Riyadh → Prioritizes GCC / Vision 2030 experience  
- 🇮🇳 Hyderabad → Prioritizes Startup & Product experience
